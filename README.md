# EtCare - Health care App

   *Care that fits your life*

## Project Description

**Etcare** is a unified health platform designed for Ethiopia. It brings together essential healthcare services into a single mobile application, making quality care accessible, trustworthy, and tailored to users’ lives.

The app offers five integrated features: an **AI Symptom Checker** that provides first‑line guidance; **telemedicine with e‑prescriptions** for remote consultations with licensed doctors; an **offline‑first health library** with articles and videos in Amharic and English; **medication adherence with gamification** to encourage consistent pill‑taking; and **personalized wellness plans** that adapt to individual health goals.
